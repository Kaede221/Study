# 关于本C++学习项目

使用Vs Studio 2022构建, 排除了`.vs`目录

- 根目录: 学习文件储存的位置, 为了方便归档, 直接储存在这里了
- 如果你发现有些文件无法运行, 请查看是否为*排除了的文件* -> 再次运行时请添加回项目后运行(包括在项目中) -> 此步骤的目的是防止一个程序中出现多个main函数后报错
- 不过啊, 目前在我的电脑上, 所有文件, 不管是否排除, 都是可以直接运行的
